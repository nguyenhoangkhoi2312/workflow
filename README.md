# Workflow UI

This is the front-end user interface for the Workflow platform (styled as Omilearn). It is an Electron app built using React and Vite.

## Features
- Document Viewer & Explorer
- Contextual Studio Sidebar
- Interactive Exam Taking & Review Interface
- Automated Study Document Generation Progress Mockups

## Running locally

```bash
npm install
npm run dev
```

To build the Electron app:
```bash
npm run electron:build
```

## [Unreleased]
### Added
- Initial commit of the Workflow UI project.
- Three-column layout architecture (Sidebar, Workspace, Studio).
- Document Viewer and Document Explorer interfaces.
- `StudioSidebar` with context-aware actions.
- `UploadModal`, `CreateExamModal`, and `StudyDocProgressModal`.
- Interactive `ExamViewer` interface with Trắc nghiệm (Multiple Choice) and Điền khuyết (Short Answer) question types.
- "Instant Reveal" (Hiện đáp án & Giải thích luôn) functionality in the `ExamViewer`.
- Split-pane layout for Exam taking with source document cross-referencing.
- Basic Electron and Vite configuration.
